DGP Sandbox
Український сандбокс сервер
Інформація


Правила

1. Не можна ображати інших гравців та їхніх рідних ( Мут, Гаг та Джайл на 10 хвилин )
2. Не можна спамити пропами, ентіті, технікою та іншими речами ( Джайл на 15 хвилин )
3. Не можна проп-пушити та проп-кілити ( Вбивати за допомогою гравіті гана можна ) ( Джайл на 10 хвилин )
4. Не можна крашити сервер ( Бан назавжди )
5. Не можна розповсюджувати анти-українські та про-російські ідеї ( Бан назавжди )
6. Не можна заважати гравцям які знаходяться у білд моді навіть коли ви знаходитесь у білд моді ( Джайл на 10 хвилин / Якщо людина в пвп це спеціально робила та бачила, що гравець у білд моді )
7. Не можна заважати гравцям які знаходяться у пвп, якщо ви у білд моді ( Джайл на 10 хвилин )
8. На сервері заборонений 18+ контент ( Джайл на 10 хвилин )
9. Не можна виходити з гри, якщо ви знаходитесь у джайлі ( Бан на 5 годин )
10. Не можна накручувати здоров'я ( Джайл на 15 хвилин )
11. Не можна обманювати модерацію сервера ( Бан на 3 години )
12. Не можна рекламувати інші сервера ( Бан назавжди )
13. Не можна використовувати PAC3, Wiremod, Outfitter та E2 для того, щоб отримати перевагу над іншими гравцями ( Бан на 1 день )
ㅤ13.2 - PAC3 можна, але заборонено робити себе дуже великим або маленьким.
ㅤ13.3 - Wiremod можна, але заборонено робити такі речі, які можуть вбивати одразу, телепортувати вас до гравця та так далі.
ㅤ13.4 - Не можна ставити полу прозорі або прозорі модельки гравця за допомогою Outfitter або PAC3
ㅤ13.5 - E2 можна, але заборонено використовувати його для того, щоб отримати перевагу над іншими гравцями.
14. Не можна використовувати стороннє ПО (програмне забезпечення) - чіти та так далі ( Бан назавжди )
15. Не можна спавнити Health kit або Suit battery під час перестрілки з іншим гравцем. Якщо ви забігли за стіну або в укриття, то хілитися можна. ( Джайл на 10 хвилин )
16. Треба обов'язково ставити кейпад для входу на базу та виходу з неї ( Джайл на 10 хв )
17. Москаль = бан
18. Спроба крашу серверу ( Бан на 2 неділі )
19. Поважаєш совок, комуняка та так далі = Бан назавжди
20. Не можна спавн-кілити ( Джайл на 15 хвилин )
21. Не можна підвішувати більше 4 бомб з АСЕ на літак, дрон та іншу техніку ( Джайл на 20 хвилин )
22. Не можна підвішувати бомби більше 500кг з АСЕ на літак, дрон та іншу техніку ( Джайл на 20 хвилин )
23. Не можна літати по мапі 406мм гарматою з АСЕ ( Джайл на 20 хвилин )
24. Не можна використовувати маленькі моделі гравця ( Джайл на 10 хвилин )
25. Не можна спавнити дублікати матеріалів, бомб та інших речей з Jmod'у ( Джайл на 20 хвилин )
26. Не можна літати на пропах ( Джайл на 10 хвилин )
27. Спавн абуз-забороняється під час перестрілок спеціально забігати в зону спавну, також забороняється використовувати зону спавну для керування дронами та іншими видами техніки які не потребують знаходитися у них, також забороняється використовувати АСЕ зброю з зони спавну щоб нищити техніку гравців ( Джайл на 10 хвилин )
